–ï—Å–ª–∏ $\ker \mathcal{A} = V$, —Ç–æ –∂–æ—Ä–¥–∞–Ω–æ–≤–∞ –Ω–æ—Ä–º–∞–ª—å–Ω–∞—è —Ñ–æ—Ä–º–∞ - –Ω—É–ª–µ–≤–∞—è –º–∞—Ç—Ä–∏—Ü–∞, –∏ –∂–æ—Ä–¥–∞–Ω–æ–≤—ã–º —è–≤–ª—è–µ—Ç—Å—è –ª—é–±–æ–π –±–∞–∑–∏—Å. $\mathcal{A}$ - –≤—ã—Ä–æ–∂–¥–µ–Ω, –∞ –∑–Ω–∞—á–∏—Ç $\ker \mathcal{A} \ne \{0\}$.(–ü–æ—á–µ–º—É?)

–ò–Ω–¥—É–∫—Ü–∏—è –ø–æ $n$:

(–ò —á—Ç–æ —Å —Ç–æ–≥–æ?) –ü—É—Å—Ç—å –∫–ª–µ—Ç–æ–∫ —Å $0$ –Ω–∞ –¥–∏–∞–≥–æ–Ω–∞–ª–∏ $s$. –ñ–æ—Ä–¥–∞–Ω–æ–≤ –±–∞–∑–∏—Å –º–∞—Ç—Ä–∏—Ü—ã $\operatorname{J}_{\operatorname{img}\mathcal{A}}$ - —ç—Ç–æ –æ–±—ä–µ–¥–∏–Ω–µ–Ω–∏–µ –∂–æ—Ä–¥–∞–Ω–æ–≤—ã—Ö –±–∞–∑–∏—Å–æ–≤ –µ–µ –∫–ª–µ—Ç–æ–∫. –ó–∞–ø–∏—à–µ–º –∂–æ—Ä–¥–∞–Ω–æ–≤ –±–∞–∑–∏—Å –¥–ª—è $\mathcal{A}_{|\operatorname{img} \mathcal{A}}$ $e = \{e_{11}, \ldots, e_{1k_1}, \ldots, e_{s1}, \ldots, e_{sk_s}, \ldots, e_{s+1 1}, \ldots, e_{s + 1 k_{s + 1}}, e_{m1}, \ldots, e_{mk_m}\}$, –≥–¥–µ $k_1, \ldots, k_m$ - —Ä–∞–∑–º–µ—Ä—ã –∂–æ—Ä–¥–∞–Ω–æ–≤—ã—Ö –∫–ª–µ—Ç–æ–∫, $\lambda_1 = \ldots = \lambda_s = 0$, $m$ - –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∂–æ—Ä–¥–∞–Ω–æ–≤—ã—Ö –∫–ª–µ—Ç–æ–∫. –ü—É—Å—Ç—å $V_0$ –ø–æ–¥–ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ –≤ $\operatorname{img} \mathcal{A}$ —Å —Å–æ–±—Å—Ç–≤–µ–Ω–Ω—ã–º –∑–Ω–∞—á–µ–Ω–∏–µ–º –Ω–æ–ª—å, –∏–º–µ–µ–º: $V_0 = \{v \in V: \,\,\,\, \mathcal{A}(v) = 0 \cdot v\} = \ker \mathcal{A}$. –¢–æ –µ—Å—Ç—å $\operatorname{img} \mathcal{A} \cap \ker \mathcal{A} = V_0$. –†–∞–∑–º–µ—Ä–Ω–æ—Å—Ç—å $V_0$ —Ä–∞–≤–Ω–∞ –∫–æ–ª–∏—á–µ—Å—Ç–≤—É –∫–ª–µ—Ç–æ–∫ —Å $0$ –Ω–∞ –¥–∏–∞–≥–æ–Ω–∞–ª–∏, —Ç–æ –µ—Å—Ç—å $s$. –í–µ–∫—Ç–æ—Ä—ã $e_{12}, \ldots, e_{s2}$ –ª–∏–Ω–µ–π–Ω–æ –Ω–µ–∑–∞–≤–∏—Å–∏–º—ã, –∞ —Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ —è–≤–ª—è—é—Ç—Å—è –±–∞–∑–∏—Å–æ–º $V_0$. –î–æ–ø–æ–ª–Ω–∏–º $e_{12}, \ldots, e_{s2}$ –≤–µ–∫—Ç–æ—Ä–∞–º–∏ $f_1, \ldots, f_q$ –¥–æ –±–∞–∑–∏—Å–∞ $f$ –ø–æ–¥–ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–∞ $\ker \mathcal{A}$.
$$\dim (\operatorname{img} \mathcal{A} \cap \ker \mathcal{A}) + \dim(\operatorname{img} \mathcal{A} + \ker \mathcal{A}) = \dim \operatorname{img} \mathcal{A} + \dim \ker \mathcal{A}$$
$$\dim \operatorname{img} \mathcal{A} + \dim \ker \mathcal{A} = n$$
$$\Downarrow$$
$$\dim (\operatorname{img} \mathcal{A} + \ker \mathcal{A}) = n - s$$
–¢–∞–∫ –∫–∞–∫ $\operatorname{img} \mathcal{A} + \ker \mathcal{A}$ - —ç—Ç–æ –ª–∏–Ω–µ–π–Ω–∞—è –æ–±–æ–ª–æ—á–∫–∞ –≤–µ–∫—Ç–æ—Ä–æ–≤ –∏–∑ $f$ –∏ $e$, –∫–æ—Ç–æ—Ä—ã–µ —è–≤–ª—è—é—Ç—Å—è –ª–∏–Ω–µ–π–Ω–æ –Ω–µ–∑–∞–≤–∏—Å–∏–º—ã–º–∏, –∞ —Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ –±–∞–∑–∏—Å–æ–º $\operatorname{img} \mathcal{A} + \ker \mathcal{A}$, –ø–æ–ª—É—á–∞–µ–º, —á—Ç–æ –Ω—É–∂–Ω–æ –¥–æ–±–∞–≤–∏—Ç—å $s$ –≤–µ–∫—Ç–æ—Ä–æ–≤ –∫ $f \cup e$, —á—Ç–æ–±—ã –ø–æ–ª—É—á–∏—Ç—å –±–∞–∑–∏—Å $V$. –î–æ–±–∞–≤–∏–º –≤–µ–∫—Ç–æ—Ä—ã $g = \{g_1, \ldots, g_s\}$:

–ö–æ–ª–∏—á–µ—Å—Ç–≤–æ –≤–µ–∫—Ç–æ—Ä–æ–≤ –≤–µ—Ä–Ω–æ–µ, –æ—Å—Ç–∞–ª–æ—Å—å –¥–æ–∫–∞–∑–∞—Ç—å –ª–∏–Ω–µ–π–Ω—É—é –Ω–µ–∑–∞–≤–∏—Å–∏–º–æ—Å—Ç—å. –†–∞—Å—Å–º–æ—Ç—Ä–∏–º –ª–∏–Ω–µ–π–Ω—É—é –∫–æ–º–±–∏–Ω–∞—Ü–∏—é —Ä–∞–≤–Ω—É—é $0$:
